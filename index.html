ChannelSort Privacy Policy

Last updated: July 7, 2026

ChannelSort is a browser extension that lets you sort a YouTube channel's videos by view count within a time range you choose. This page explains what data the extension touches and what it does with it.

What ChannelSort stores

The only piece of data ChannelSort stores is your own YouTube Data API v3 key, which you obtain directly from Google and paste into the extension. This key is stored using Chrome's built-in sync storage, tied to your own Google account. It is never transmitted to us, and we do not operate any server that receives it.

What ChannelSort sends, and to whom

  • To Google's YouTube Data API: when you use ChannelSort, it sends requests directly from your browser to Google's YouTube Data API v3, using your own API key, to retrieve a channel's video list, view counts, and publish dates. This traffic goes straight from your browser to Google, not through us.
  • To PayPal: if you choose to click the optional donation link, your browser navigates to PayPal's donation page. We do not receive or see any information about that transaction.

ChannelSort does not run its own backend server, and does not send your API key, browsing activity, or any other data to us or to any third party other than the two cases above, both of which only happen as a direct result of an action you take.

What ChannelSort does not do

  • It does not collect analytics, usage statistics, or telemetry of any kind.
  • It does not track which channels or videos you view or sort.
  • It does not sell, share, or transmit your data to advertisers or any other third party.

Permissions this extension requests, and why

  • storage — to save your API key locally/synced via Chrome, so you don't have to re-enter it every time.
  • activeTab — to read the current tab's URL so the extension can identify which channel you're viewing.
  • contextMenus — to add the optional "Donate" entry to the extension icon's right-click menu.
  • Host access to youtube.com — to detect channel pages and inject the ChannelSort option into the sort bar.
  • Host access to googleapis.com — to call the YouTube Data API v3 using your key.
